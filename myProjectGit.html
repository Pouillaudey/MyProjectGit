Mes premiers pas avec Git

Git

Base de Git :

  • Installer GIT
  • Se rendre sur la console
  • Créer un repertoire mkdir monRepositoryGit
  • Se rendre dans le repertoire cd monRepository
  • Dans le repertoire, pour utiliser le versionning, faire git init
  • Créer un fichier que vous voulez utiliser en versionning
  • Enfin pour enregistrer ce fichier dans votre repository, faire git add fichier.extension
  • Votre repertoire et votre fichier sont initialisé
  • Maintenant pour enregistrer tous ça, faire git commit -m "Explication de la sauvegarde"
